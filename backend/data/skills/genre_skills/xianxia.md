# 仙侠风格写作指导

## 核心要素

1. **修炼体系**：明确的境界划分（炼气→筑基→金丹→元婴→化神→大乘→渡劫）
2. **世界观**：凡界/灵界/仙界多层世界结构，灵气浓度决定地域等级
3. **核心冲突**：资源争夺、宗门恩怨、正邪对抗、天道争锋
4. **角色类型**：天赋异禀的主角、高深莫测的师父、势大的反派宗门

## 写作要点

- 修炼突破场景要有仪式感和视觉冲击力
- 法术战斗描写要有层次感，从气势→招式→胜负
- 机缘奇遇要合理铺垫，避免强行开挂
- 人物对话可半文半白，显示修真界独特语境
- 速度节奏：前期快节奏展示升级快感，中后期深化世界观
