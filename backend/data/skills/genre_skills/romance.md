# 言情风格写作指导

## 核心要素

1. **情感弧线**：初遇→好感→矛盾→分离→重逢/决断
2. **人物配对**：性格互补或相似，制造化学反应
3. **冲突来源**：身份差距、误会、第三方、三观差异
4. **场景设计**：日常互动（60%）+ 关键事件（30%）+ 独白（10%）

## 写作要点

- 感情发展要有节奏感，不能太快也不能太拖
- 内心独白与外部行动相结合表现情感
- 对话中的潜台词比直白表白更有张力
- 肢体语言的细节描写（不经意触碰、眼神游离）
- 虐要有原因，甜要有铺垫
