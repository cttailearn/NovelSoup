# 悬疑风格写作指导

## 核心要素

1. **谜题设计**：核心谜题（案件/秘密）需有公平线索
2. **信息控制**：读者与侦探的信息差，何时揭示线索
3. **节奏控制**：调查推进（快）与线索整合（慢）交替
4. **干扰项**：合理设置红鲱鱼（迷惑线索）

## 写作要点

- 线索要提前布置但不显眼（埋得好，收得妙）
- 每章结尾留一个小悬念或新发现
- 反转要有逻辑支撑，不能为反转而反转
- 环境描写渲染紧张气氛（天气、时间、空间）
- 通过不同角色提供碎片信息，拼凑真相
