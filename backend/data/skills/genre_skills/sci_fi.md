# 科幻风格写作指导

## 核心要素

1. **科技设定**：核心科技概念（AI、量子、太空、基因）需有合理依据
2. **世界观**：未来社会结构、科技对生活的影响、新伦理问题
3. **核心冲突**：人与科技、人与异族、旧秩序与新变革
4. **叙事视角**：宏观（文明史诗）与微观（个人命运）结合

## 写作要点

- 科技概念用通俗类比解释，避免信息堆砌
- 未来世界观通过细节自然展现，而非大段说明
- 科幻设定服务于情节和主题，不是炫技
- 保持科学素养，核心设定有现实科学基础
- 即便在未来系，人类情感和道德困境依然是核心
