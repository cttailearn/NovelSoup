# 都市风格写作指导

## 核心要素

1. **现实质感**：场景要接地气（写字楼、地铁、出租屋、商圈）
2. **人物关系**：职场、家庭、朋友圈多层关系交织
3. **时代印记**：融入当下社会热点和生活方式
4. **核心冲突**：理想与现实、阶层差距、道德选择

## 写作要点

- 场景描写要具体可感，唤起读者日常共鸣
- 对话自然口语化，不同阶层用不同语言风格
- 专业领域要有扎实的背景调研
- 生活细节的质感比宏大叙事更重要
- 人物困境要真实，解决方式不能太过理想化
