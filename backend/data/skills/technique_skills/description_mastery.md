# 描写技法大全

## 环境描写技法

### 气氛渲染法
通过天气、光线、声音、气味渲染特定氛围。
- 悲伤：阴雨天、黄昏、枯叶
- 紧张：狭窄空间、倒计时、黑暗
- 温馨：暖光、咖啡香、轻音乐

### 细节聚焦法
选择一个代表性的细节反复描写，以小见大。
- 如：《孔乙己》中的"茴"字

### 多感官综合法
同时描写视觉、听觉、嗅觉、触觉、味觉中的至少三种。

## 人物描写技法

### 间接描写法
通过他人的反应侧面烘托人物特点。
### 动态描写法
通过人物的动作和选择来揭示性格。
### 对比描写法
通过与其他人物的对比来突出特点。

## 动作描写技法

### 分解动作法
将一个大动作分解为一系列小动作。
### 节奏控制法
用句子长短控制动作的紧迫感。
