# 情感场景写作技法

## 核心原则

1. **克制胜于煽情**：留白让读者自己体会比直接抒情更有力
2. **细节见真情**：一个动作胜过千言万语
3. **环境烘托**：用环境映射内心状态
4. **节奏变化**：情感高点前后的节奏变化制造张力

## 情感类型与技法

### 表白场景
- 紧张感通过生理反应（心跳、手汗、口干）间接表现
- 对话要有犹豫和停顿，不是一气呵成
- 回应可以有延迟，制造悬疑感

### 离别场景
- 凝视细节（最后一次看对方的脸，记住某个细节）
- 背景人潮/列车等动态元素对比静态的离别
- 未说的话比说出的话更有分量

### 背叛场景
- 先用细节暗示异常，再揭穿真相
- 被背叛者的情感变化：震惊→否认→愤怒→悲伤
