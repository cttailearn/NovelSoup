# 打斗场景写作技法

## 核心原则

1. **节奏控制**：短句加速紧张感，长句用于喘息和战术思考
2. **空间明确**：让读者清楚知道谁在哪，武器在哪
3. **感受优先**：不只写动作，更要写身体的感受（疼痛、疲劳、肾上腺素）
4. **胜负设计**：胜负要有说服力，不能靠运气

## 描写层次

- **宏观层（10%）**：战场整体态势
- **中观层（30%）**：双方招式和应对
- **微观层（60%）**：具体感受（气劲冲击、骨骼断裂声、血腥味）
