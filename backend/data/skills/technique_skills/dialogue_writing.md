# 对话写作技法

## 核心原则

1. **对话即行动**：每句对话都应该推动情节或揭示人物
2. **一人一腔**：不同角色有独特的说话方式
3. **潜台词**：真正想说的往往没说出口
4. **节奏变化**：通过对话长度控制叙事节奏

## 对话设计技巧

- **语癖设计**：每个主要角色有独特的口头禅或说话习惯
- **信息差**：利用角色知道的信息不同制造戏剧张力
- **打断与停顿**：用省略号和破折号表示迟疑和打断
- **沉默胜于千言**：有时不说话比说话更有力量

## 对话与描写结合

- 对话前后穿插动作描写（"他说" → "他放下茶杯，说"）
- 通过对话时的表情和动作透露真实情绪
- 环境描写与对话节奏配合（紧张时用短句，舒缓时用长句）
