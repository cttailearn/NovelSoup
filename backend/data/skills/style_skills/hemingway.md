# 海明威风格写作指导

## 核心原则

1. **冰山理论**：只写出水面上的八分之一，水下的八分之七留给读者
2. **简洁对话**：人物对话简短有力，避免冗长独白
3. **动作驱动**：用人物的动作暗示内心，而非直接描写心理
4. **短句为王**：多用短句，少用复合句，避免形容词堆砌
5. **具体而非抽象**：用具体的事物和细节来传达情感

## 写作特点

- 对话占主导地位，推动情节发展
- 叙述语言高度克制，几乎没有主观评论
- 重复使用相同的简单词汇（如"said"）
- 描写环境时注重光线、天气等客观元素
- 节奏快速，没有冗长的心理描写

## 句式要求

- 每句话尽量不超过20个字
- 对话使用简单的"说"、"问"等引导词
- 段落短小精悍，重要内容独立成段
- 避免使用比喻、拟人等修辞手法
