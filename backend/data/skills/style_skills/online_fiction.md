# 网文风格写作指导

## 核心原则

1. **爽点密集**：每章至少一个让读者感到"爽"的点
2. **快节奏**：减少不必要的环境描写，快速推进情节
3. **代入感强**：多用内心独白和第一人称感受描写
4. **金句频出**：每章要有1-2句令人印象深刻的台词
5. **悬念收放**：章末留悬念，但要在3章内解决

## 写作特点

- 每章3000-5000字，段落短小，方便手机阅读
- 开篇3章内必须出现核心冲突
- 主角要有明确的成长线和目标
- 配角功能明确，避免过于复杂的支线
- 对话白话化，接近日常口语

## 节奏把控

- 前500字：承接上章悬念
- 500-2000字：情节推进
- 2000字-结尾：高潮+新悬念
